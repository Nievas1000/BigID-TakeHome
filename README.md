---

### **Steps to Run the Application**

#### **1. Clone the Repository**
```bash
git clone https://github.com/Nievas1000/BigID-TakeHome.git
cd BigID-TakeHome
```

#### **2. Set Up the Backend**
```bash
cd backend
npm install
npm start
```
#### **3 Set Up the Frontend**
```bash
cd frontend
npm install
npm run dev
```
